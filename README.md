# Flutter Projesi – Mini Katalog Uygulaması

## Proje Adı

Flutter Projesi – Mini Katalog Uygulaması

## Proje Açıklaması

Bu proje Flutter kullanılarak geliştirilmiş basit bir mobil katalog uygulamasıdır.
Uygulamada ürünler listelenir ve kullanıcılar ürünleri sepete ekleyebilir.
Proje, Flutter eğitim sürecinde widget yapısı, listeleme, ekranlar arası geçiş ve temel kullanıcı arayüzü geliştirme konularını öğrenmek amacıyla hazırlanmıştır.

## Kullanılan Teknolojiler

* Flutter
* Dart
* Material Design

## Flutter Sürümü

Flutter 3.x

## Uygulama Özellikleri

* Ürün listesini görüntüleme
* Ürün detay sayfası
* Sepete ürün ekleme
* Sepet ekranında seçilen ürünleri görüntüleme

## Projeyi Çalıştırma Adımları

1. Projeyi bilgisayarınıza klonlayın

```
git clone https://github.com/KULLANICI_ADIN/Flutter_Projesi.git
```

2. Proje klasörüne girin

```
cd flutter_application
```

3. Gerekli paketleri yükleyin

```
flutter pub get
```

4. Uygulamayı çalıştırın

```
flutter run
```
